About

HTML Link Replacer is a user-friendly desktop application that helps you quickly update multiple HTML links across your files. Whether you're migrating a website, updating URLs, or fixing broken links, this tool makes the process simple and efficient.

Installation Guide

  1. Download the HTML Link Replacer.dmg file by clicking the download button above
  2. Double-click the downloaded DMG file to open it
  3. Drag the HTML Link Replacer application to your Applications folder
  4. Eject the disk image (drag it to the trash or right-click and select "Eject")
  5. Open your Applications folder and double-click "HTML Link Replacer" to run it
Note: When opening the application for the first time, you might see a security warning because the app is from an unidentified developer. To open it anyway:
  1. Right-click (or Control-click) the app in the Applications folder
  2. Choose "Open" from the shortcut menu
  3. Click "Open" in the dialog box
After this first time, you can open it normally by double-clicking.

How to Use

  1. Launch HTML Link Replacer from your Applications folder
  2. Click "Select Files" to choose the HTML files you want to process
  3. Enter the old link pattern you want to replace
  4. Enter the new link you want to use instead
  5. Click "Replace Links" to process your files
  6. Review the changes in your HTML files

System Requirements

  • macOS 10.13 or later
  • 64-bit processor
  • 50MB of free disk space
